:) Annotations
    -> It is form of metadata, provide data of program but not the part of program itself.
    -> Uses:
        i) info for the compiler
        ii) Compile-time and deployment time processing.
        iii) Runtime processing.

    -> @ (it indicates to compiler that it will follow by an annotations.)
    -> Types of annotations(custom):
        i)Maker annotations.
        ii)Single-value annotations.
        iii)Multiple-value annotations.

    -> Built-in annotations used with custom annotations:
        @Target
        @Retention
        @Inherited
        @Documented

        use of above-mentioned annotations:


    -> Built-in annotations used with java code.
        @Deprecated
        @SuppressWarnings
        @Override
